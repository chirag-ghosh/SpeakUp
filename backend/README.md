# <div align="center">SpeakUp FastAPI backend</div>

<br/>

## Tech Stack

**Python version:** 3.10

**Framework:** FastAPI

<br/>

## Run Locally

Clone the project

```bash
  git clone https://github.com/chirag-ghosh/SpeakUp
```

Go to the project backend directory

```bash
  cd SpeakUp/backend
```

Install dependencies

```bash
  pipenv install
```

Start the server

```bash
  pipenv shell
  uvicorn app.main:app --reload
```
